# react-typescript-library-template

## Install
```
yarn install
```

## Run Tests
```
yarn test
```

## Run Storybook
```
yarn storybook
```

## Usage

```tsx
import * as React from 'react';

import { Hello } from 'react-typescript-library-template';

class Example extends React.Component {
  render () {
    return (
      <Hello text="hello" />
    )
  }
}
```
